# Airport Operations Intelligence Platform — Datasets (Project 5 - Skyhaven)

Generated by `generate_airport_analytics.py`. Use for Bronze/Silver/Gold pipeline, OTP report, delay propagation, baggage misconnect risk, gate utilization, and security staffing.

**Total: ~20,400 rows** across 6 sources.

## Layout

| Path | File | Rows | Format | Key Fields |
|------|------|------|--------|------------|
| **flight_schedule/** | flight_schedule.csv | 400 | CSV | flight_id, flight_number, airline_code, airline_name, origin_iata, destination_iata, scheduled_departure, scheduled_arrival, aircraft_registration, aircraft_type, terminal, gate, status (SCH\|BRD\|DEP\|ARR\|DLY\|CNX), delay_minutes, delay_reason_code, pax_count_booked |
| **gate_events/** | gate_events.json | 2,500 | NDJSON | event_id, flight_id, gate_id, terminal, event_type (GATE_ASSIGNED\|GATE_OPEN\|BOARDING_START\|BOARDING_COMPLETE\|PUSHBACK\|GATE_CLOSED\|GATE_RELEASED), event_ts, agent_id, remarks |
| **baggage_scans/** | baggage_scans.csv | 12,000 | CSV | scan_id, bag_tag_number, passenger_id, flight_id, scan_location (CHECK_IN\|SCREENING\|MAKEUP_BELT\|AIRCRAFT_LOAD\|AIRCRAFT_UNLOAD\|CAROUSEL\|TRANSFER_BELT), scan_ts, scan_status (OK\|EXCEPTION\|MISREAD), connecting_flight_id |
| **passenger_counters/** | passenger_counters.json | 5,050 | NDJSON | sensor_id, location_type (SECURITY_LANE\|IMMIGRATION\|GATE_AREA\|CHECK_IN_ZONE\|BAGGAGE_CLAIM), location_name, terminal, zone_id, timestamp, pax_count_in, pax_count_out, queue_length_estimate, avg_wait_time_minutes |
| **airline_sla/** | airline_sla_contracts.xlsx, .csv | 50 | XLSX/CSV | airline_code, airline_name, min_turn_time_narrow_body_mins, min_turn_time_wide_body_mins, connection_time_domestic_mins, connection_time_international_mins, late_departure_penalty_inr_per_min, otp_target_pct |
| **aircraft_master/** | aircraft_master.csv | 400 | CSV | aircraft_registration, aircraft_type_code, body_type (NARROW\|WIDE), airline_code, seat_capacity, max_payload_kg, age_years, base_airport_iata |

## Referential integrity

- **flight_schedule.aircraft_registration** exists in **aircraft_master.aircraft_registration**
- **flight_schedule.airline_code** exists in **airline_sla_contracts.airline_code**
- **gate_events.flight_id** and **baggage_scans.flight_id** reference **flight_schedule.flight_id**

## Regenerate

From repo root:
```bash
python generate_airport_analytics.py
```
Optional: `pip install pandas openpyxl` for airline_sla XLSX.

## Pipeline use

- **Bronze**: Flight schedule (MERGE on flight_id), gate events (append), baggage (append), IoT (append), SLA + aircraft (reference).
- **Silver**: flight_operations (join schedule + gate events), delay_propagation_risk (turn time vs SLA), baggage_journey + misconnect risk, zone_occupancy_15min (IoT aggregation).
- **Gold**: otp_daily_report (DGCA), gate_utilization_summary, security_staffing_recommendation.
